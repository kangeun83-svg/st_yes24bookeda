# Streamlit 대시보드 생성 프롬프트

**역할**: 전문 데이터 분석가 및 Streamlit 개발자
**목표**: `book_data.csv`를 시각화하고 분석하기 위한 포괄적인 Streamlit 대시보드를 생성하세요. 대시보드는 상호작용 가능하고, 통찰력을 제공하며, 사용자 친화적이어야 합니다. 모든 시각화에는 **Plotly**를 사용하세요.

## 1. 대시보드 구조 및 레이아웃

-   **라이브러리**: 앱에는 `streamlit`, 대화형 차트에는 `plotly`를 사용하세요.
-   **사이드바 메뉴**: 내비게이션을 위해 최소 **5개의 메뉴 항목**이 있는 사이드바를 구현하세요 (예: 홈, 판매 분석, 출판사 분석, 가격 및 평점, 키워드 검색).
-   **탭**: 메인 분석 페이지 내부에는 `st.tabs`를 사용하여 콘텐츠를 구성하고, 해당되는 경우 섹션당 **최소 3개의 탭**을 두세요.
-   **레이아웃**: 깔끔하고 전문적인 레이아웃을 보장하세요. 각 페이지에는 **최소 2개의 별도 시각화**가 포함되어야 합니다.

## 2. 기능 요구사항

### 2.1 데이터 로드 및 전처리
-   `book_data.csv`를 로드하세요.
-   결측치 및 데이터 타입 변환을 처리하세요 (Price, Rating, Review Count, Sales Index).
-   시계열 분석을 위해 날짜(`Publishing Date`)를 파싱하세요.

### 2.2 핵심 기능
1.  **판매지수 분석 (Sales Index Analysis)**:
    -   판매지수 상위 20개 도서를 시각화하세요.
    -   카테고리 또는 출판사별 판매지수 분포를 비교하세요.
2.  **출판사 검색 및 분석 (Publisher Search & Analysis)**:
    -   사용자가 드롭다운에서 **출판사를 검색/선택**할 수 있게 하세요.
    -   선택된 출판사에 대한 통계(평균 가격, 총 도서 수, 평균 평점)를 보여주세요.
    -   이 출판사에서 출간된 도서의 시간 흐름에 따른 트렌드를 시각화하세요.
3.  **키워드 검색 (Keyword Search)**:
    -   제목이나 주요 주제로 도서를 필터링할 수 있는 **키워드 검색창**을 구현하세요.
    -   필터링된 결과를 데이터 테이블로 표시하고, 가능하다면 해당 하위 집합에 대한 워드클라우드를 생성하세요.

## 3. 시각화 및 설명 요구사항

-   **Plotly 전용**: 모든 차트는 `plotly.express` 또는 `plotly.graph_objects`를 사용하여 상호작용(확대/축소, 호버 등)이 가능해야 합니다.
-   **상세 설명**:
    -   **각 시각화**에 대해 **최소 200자**의 설명 텍스트 블록을 제공하세요.
    -   차트가 *무엇을* 보여주는지, *왜* 중요한지, 어떤 *인사이트*를 얻을 수 있는지 설명하세요 (예: "가격 히스토그램을 보면 대부분의 IT 도서가 20,000원에서 30,000원 사이 구간에 분포하고 있음을 알 수 있으며...").
-   **보조 데이터 테이블**:
    -   **모든 시각화 하단**에 해당 차트의 기반 데이터를 나타내는 **피벗 테이블** 또는 **교차표(Crosstab)**를 표시하세요.
    -   이는 사용자가 시각적 데이터를 정확한 수치로 확인할 수 있게 돕습니다.

## 4. 특정 페이지 및 콘텐츠 아이디어

### 페이지 1: 개요 (Overview / Home)
-   주요 지표 (총 도서 수, 평균 가격, 총 리뷰 수).
-   최근 트렌드 (시간에 따른 출판 수 선 그래프).
-   *설명 및 데이터 테이블 포함.*

### 페이지 2: 판매 및 랭킹 (Sales & Ranking)
-   판매지수 상위 20개 막대 차트.
-   산점도: 판매지수 vs 리뷰 수.
-   *설명(200자 이상) 및 각 차트별 피벗 테이블.*

### 페이지 3: 출판사 인사이트 (Publisher Insights)
-   출판사 선택 드롭다운.
-   선택된 출판사에 대한 지표.
-   차트: 이 출판사의 가격 분포, 평점 분포.
-   *설명(200자 이상) 및 교차표.*

### 페이지 4: 가격 및 평점 분석 (Price & Rating Analysis)
-   상위 출판사별 가격 박스 플롯.
-   상관관계 히트맵 (가격, 평점, 리뷰 수, 판매지수).
-   *설명(200자 이상) 및 데이터 테이블.*

### 페이지 5: 키워드 및 검색 (Keyword & Search)
-   제목의 워드클라우드 (전체 또는 필터링됨).
-   일치하는 도서의 상세 테이블을 보여주는 검색창 기능.
-   *설명 및 요약 테이블.*

## 5. 결과물
-   전체 Streamlit 코드가 포함된 단일 Python 파일 (예: `dashboard.py`).
-   `streamlit run dashboard.py`로 바로 실행 가능하도록 코드가 준비되어야 함.
